# Inventory Managament

## Installation
#### 1. Goto dbserver and Run
```bash
npm install
```
#### 2. Goto appserver/e-commerce and Run
```bash
npm install
```
#### 3. Setup mongo db and node on your system

## Run
#### 1. Goto dbserver and Run
```bash
node server.js
```
#### 2. Goto appserver/e-commerce and Run
```bash
ng serve 
```

#### 3. App will be starts on localhost:4200
